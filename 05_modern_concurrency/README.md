# Module: Modern Concurrency and Parallelism

## Description

Learn to write concurrent and parallel Python code. This module covers threading, multiprocessing, asyncio, and modern Python runtime features such as free-threading and subinterpreters.

**Key concepts**: asyncio, coroutines, threading, multiprocessing, GIL, free-threading, subinterpreters, TaskGroups.

## Learning Objectives

- Understand the tradeoffs between concurrency models in Python
- Build correct programs with threads, processes, and async tasks
- Apply synchronization primitives and coordination patterns safely
- Recognize runtime-level implications of modern concurrency features
