# Module: Basic and Intermediate OOP

## Description

Master Object-Oriented Programming in Python. From classes and objects to inheritance, properties, descriptors, and dataclasses, this module helps you write Pythonic and maintainable OOP code.

**Key concepts**: Classes, objects, attributes, methods, inheritance, polymorphism, encapsulation, special methods, properties, descriptors, composition.

## Learning Objectives

- Understand the foundations of object-oriented design in Python
- Build reusable models with encapsulation and polymorphism
- Apply Python-specific OOP tools such as dunder methods, properties, and dataclasses
