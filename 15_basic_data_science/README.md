# Module 15: Basic Data Science

> NumPy, Pandas, visualization, cleaning, EDA, and Polars for an approachable data workflow in Python.

## Description

This module introduces a practical data science flow: tabular manipulation, charting, data cleaning, early exploration, and modern tooling from the Python ecosystem.

## Learning Goals

- Understand how to structure datasets with arrays and tables.
- Build exploratory reports and simple visualizations.
- Apply data cleaning before analysis or communication.
- Compare classic tools (Pandas, Matplotlib) with modern options (Polars).
