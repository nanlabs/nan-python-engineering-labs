# Advanced Python 2026

A comprehensive module covering:

## Part 1: PyO3 and Rust-Python Integration (Topics 1-22)

Deep dive into building high-performance Python extensions using Rust with PyO3, including:
- PyO3 fundamentals and setup
- Type conversions and error handling
- Classes, methods, and properties
- Performance optimization and parallel processing
- NumPy integration and advanced patterns

## Part 2: AI-Assisted Development (Topics 23-45)

Practical patterns for building AI systems with embeddings, LLMs, and intelligent agents:
- Embeddings and vector stores
- LLM streaming and structured outputs
- LangChain and LangGraph fundamentals
- Retrieval-Augmented Generation (RAG)
- Multi-agent systems and testing
- Cost tracking and optimization

## Learning Path

Start with Part 1 for performance fundamentals, or Part 2 for AI development. Each topic is self-contained but builds on earlier concepts.

## How to Use

Each topic includes:
- **README.md**: Complete lesson with concepts, use cases, and reflection prompts
